# Lecture 56 : Layer 1: Physical Layer-II
1. **Signal-to-Noise Ratio (SNR)**  

SNR is the single most important metric for channel   quality. A high SNR means the signal is strong   relative to the noise, allowing for more reliable data transmission at higher speeds.  

2. **Shannon's Capacity Formula.**

Formula: Capacity (C) = Bandwidth (B) * log₂(1 + SNR)  

* Shannon Sets the Speed Limit: Shannon's Capacity theorem defines the theoretical maximum bit rate for any noisy channel, providing a hard physical limit.  


3. **Choose a Practical Bit Rate and Find the Required Signal Levels with Nyquist's Formula.**

* Nyquist Defines the "How": Nyquist's formula connects bit rate to the number of signal levels, showing that higher speeds on a given bandwidth require more complex (and potentially less reliable) signals.
